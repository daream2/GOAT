# GOAT: A Training Framework for Goal-Oriented Agent with Tools

Project page for the paper [GOAT: A Training Framework for Goal-Oriented Agent with Tools](https://arxiv.org/abs/2510.12218) (ACL 2026 Findings).

Built on the [Nerfies](https://github.com/nerfies/nerfies.github.io) project page template.

## Local preview

```bash
# from this folder
python3 -m http.server 8000
# then open http://localhost:8000
```

## Editing

- Edit `index.html` — title, authors, abstract, method, results, BibTeX.
- Put figures in `static/images/` and reference them, e.g. `<img src="./static/images/method.png" class="center"/>`.
- Math is rendered with KaTeX: inline `\(x\)`, display `\[ y = f(x) \]`.

## Deploy with GitHub Pages

1. Push to GitHub (the `main` branch).
2. Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Select branch `main` and folder `/ (root)`, then **Save**.
4. The site will be available at `https://daream2.github.io/GOAT/`.
